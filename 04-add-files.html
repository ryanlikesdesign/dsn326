Add Your Files

Upload your HTML, CSS, and images to your repository — all from the browser.

Two Ways to Add Files

You can either upload files from your computer or create files directly in GitHub's browser editor. We'll cover both.

Option A: Upload Files From Your Computer

This is the easiest method if you've already been working on your files locally.

  1. Go to your repository page on GitHub.
  2. Click the "Add file" dropdown button (near the top, next to the green "Code" button).
  3. Select "Upload files".
  4. Drag and drop your files into the upload area, or click "choose your files" to browse.
    Tip: You can upload multiple files and even entire folders at once. Drag a whole folder from your computer into the upload area.
  5. At the bottom of the page, you'll see a "Commit changes" section. Leave the default message or write something like "Add initial site files."
  6. Click "Commit changes".

Option B: Create a File in the Browser

This is useful for creating new files or making quick edits.

  1. Go to your repository page on GitHub.
  2. Click the "Add file" dropdown and select "Create new file".
  3. In the filename field at the top, type the name of your file — for example, index.html.
  4. In the large text editor below, type or paste your HTML code.
  5. Scroll down and click "Commit changes".
Critical: Your main page must be named index.html (all lowercase). This is the file GitHub Pages will load when someone visits your site URL. If you name it Index.html, home.html, or anything else, your site won't load properly.

Creating Folders (Subfolders)

To create a folder inside your repo (for example, a css or images folder):

  1. Click "Add file" → "Create new file".
  2. In the filename field, type the folder name followed by a slash, then the filename. For example: css/style.css. GitHub will automatically create the css folder.
  3. Add your CSS code and commit.

Your First File: A Starter index.html

If you don't have files yet, here's a minimal starter page you can use:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Design System</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>Welcome to My Design System</h1>
  <p>This site is under construction.</p>
</body>
</html>

And a matching starter style.css:

body {
  font-family: sans-serif;
  max-width: 800px;
  margin: 2rem auto;
  padding: 0 1rem;
  color: #333;
}

h1 {
  color: #1a1a2e;
}

Quick Checklist

  • I have an index.html file in the root of my repo (not inside a folder)
  • My CSS file is linked correctly in the HTML
  • My file and folder names are all lowercase with no spaces
  • I committed my changes
