What Is GitHub?

A plain-English explanation — no technical background needed.

← Back to Getting Started

The Short Version

GitHub is a free website where you can store files and publish websites. Think of it like Google Drive, but designed specifically for code and web projects. When you put your HTML and CSS files on GitHub and turn on a feature called GitHub Pages, GitHub will host your files as a real, live website that anyone can visit.

Why Not Just Use Google Drive or Dropbox?

Google Drive and Dropbox are great for storing files, but they don't turn your files into a website. If you upload an HTML file to Google Drive, you get a download link — not a working web page. GitHub Pages actually serves your files as a website, complete with a real URL you can share.

Key Terms (In Plain English)

Repository (or "repo")
A project folder on GitHub. It holds all your files for one project — your HTML pages, your CSS stylesheet, your images, everything. You'll create one repository for your design system project.
GitHub Pages
A free feature that turns a repository into a live website. You flip a switch in your repo settings, and GitHub publishes your files at a URL like yourusername.github.io/your-repo-name.
Commit
Saving a change. When you edit a file on GitHub and click "Commit changes," you're saving that version. It's like hitting Save, but GitHub also keeps a history of every version.
Branch
A version of your project. For this class, you'll only use the default branch called main. You don't need to worry about creating additional branches.

Why Designers Use GitHub

You might think GitHub is only for software engineers, but many designers use it too. Here's why it's worth learning:

  • Portfolio hosting — Your github.io URL is a professional link you can put on your resume.
  • Version history — If you break something, you can always go back to a previous version.
  • Industry standard — Many design jobs expect familiarity with GitHub, especially in product design and UX engineering roles.
  • It's free — No subscription, no trial period, no credit card required.
Bottom line: For this class, GitHub is simply the place where your website lives. You upload your files, turn on GitHub Pages, and you get a live URL. That's it.
